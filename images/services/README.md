# Service Background Images

This directory should contain background images for the service cards.

## Required Images:

1. **character.jpg** - Background for "Character & Creature Art" service card
   - Recommended size: 800x600 pixels
   - Theme: Characters, creatures, or monsters

2. **weapons.jpg** - Background for "Weapons & Vehicles" service card
   - Recommended size: 800x600 pixels
   - Theme: Weapons, vehicles, or mechanical items

3. **props.jpg** - Background for "Props & Environments" service card
   - Recommended size: 800x600 pixels
   - Theme: Props, environments, or scenes

## Format:
- JPG or PNG format
- Keep file sizes reasonable (under 500KB per image)
- Images will be displayed with an overlay, so they don't need to be too bright

## Note:
If no images are provided, the cards will display with a solid background color as fallback.
